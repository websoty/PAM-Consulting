# PAM Consulting — Многостраничный адаптивный сайт
## Описание
Многостраничный адаптивный сайт с mobile-first версткой, динамическими блоками и оптимизацией скорости загрузки.
## Демо (Vercel)
[Посмотреть проект](https://pam-consulting-fegv.vercel.app/)
## Макет Figma
[Открыть макет в Figma](https://www.figma.com/design/ePduaXLNk9ciULuPnwfuxV/PAMCONSULTING-info--Copy-?node-id=0-1&p=f&t=HHj1ejdsGyKhZky8-0)
## Основной функционал
- Асинхронная подгрузка блоков через fetch, ускоряющая загрузку страниц
- Динамическое модальное окно с формой контакта
- Слайдеры (Slick.js) с autoplay, адаптивным количеством слайдов и lazy load
- Адаптивное меню с анимацией пунктов
- Lazy loading всех крупных изображений для оптимизации скорости
- Mobile-first верстка

## Технологии
HTML5, SCSS, JavaScript (DOM, события, асинхронные запросы), Slick.js, Vercel(для деплоя)

## Что демонстрирует проект
- Создание многостраничного интерфейса с динамической подгрузкой контента
- Управление DOM: создание, удаление, обновление элементов
- Оптимизация загрузки страниц и изображений
- Разработка интерактивных элементов
- Адаптация интерфейса под мобильные и desktop устройства
